#### 说明
将一个文件从一个目录复制到另一目录，并重命名。

#### 脚本代码

```
@echo off
echo starting...

cd\
Z:
cd "vp128marsback\vendor\byd\app\CarStatus"

if exist CarStatus.apk (
    echo This is CarStatus.apk
)else (
    echo CarStatus.apk missing.
)

del CarStatus.apk
echo del CarStatus.apk

copy D:\work1\Carstatus\main\release\main-release.apk Z:\vp128marsback\vendor\byd\app\CarStatus\CarStatus.apk
echo copy CarStatus.apk

pause

```
