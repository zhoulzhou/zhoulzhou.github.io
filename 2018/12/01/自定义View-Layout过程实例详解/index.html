答：首先明确定义：

getWidth() / getHeight()：获得View最终的宽 / 高
getMeasuredWidth() / getMeasuredHeight()：获得 View测量的宽 / 高

：一般情况下，二者获取的宽 / 高是相等的

在非人为设置的情况下，View的最终宽/高（getWidth() / getHeight()）
